# Algorithms in C (CS1101)

A collection of algorithms, programming exercises, and practical implementations completed as part of the **CS1101 - Fundamentals of Programming** course. This repository documents my learning journey and serves as a reference for understanding fundamental algorithmic concepts using the C programming language.

## 📌 About

This repository contains:

* Algorithm implementations

The purpose of this repository is to strengthen my understanding of algorithms, improve programming skills, and develop efficient problem-solving approaches.

## 🚀 Topics Covered

The following topics have been completed and are included in this repository:
 
 * Linear Search
 * Bubble Sort
 * Heap Sort

> This repository will be updated as additional topics are learned and implemented.

## 💻 Requirements

* GCC Compiler (recommended)
* Any C-compatible IDE or code editor:

  * Visual Studio Code
  * Code::Blocks
  * CLion

## ▶️ Running the Programs

Clone the repository:

```bash
git clone https://github.com/Nipun-Lakshan/algorithms-in-c-cs1101.git
```

Navigate to the project directory:

```bash
cd algorithms-in-c-cs1101
```

Compile a C program:

```bash
gcc filename.c -o program
```

Run the executable:

```bash
./program
```
